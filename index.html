brunch

About

brunch is an opinionated client side framework with REST, HTML and CSS in mind.

Standing on the Shoulders of Giants *

Instead of reinventing the wheel, brunch assembles awesome wheels.

  • CoffeeScript
  • Backbone.js
  • Underscore.js
  • Stylus
  • jQuery & Zepto.js

Getting Started

You can get brunch via the nodejs package manager npm install brunch

Once installed, you can create a new project using

brunch new $proj

where $proj is the namespace you want to use for your application (e.g. 'app').

This will create the initial project structure for you. Then you can start the file-watcher using

brunch watch

It will then compile your .coffee, .styl and .eco files on the fly after every change so you don't have to do it manually.
